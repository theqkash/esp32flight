esp32flight

Standalone flight radar for the Waveshare ESP32-S3-Touch-LCD-7

Install from your browser

Connect the board over USB, click Install and pick the serial port. The full firmware (v0.1.0, 16 MB including airline logos and maps) will be written to the device.

Chrome or Edge on desktop required (Web Serial)

The board uses a CH343 USB-serial chip. If no port shows up, install the WCH driver (Windows) and try again. Flashing takes a few minutes.

What you get

  • Live aircraft within a configurable radius, with airline logos and types
  • Routes with cities, countries and local airport times, ETA, aircraft photos
  • Route map, radar view and an ambient auto-cycling mode
  • Weather in the header, emergency squawk alerts, 7 color themes, EN/PL UI
  • Web panel with live map, stats and OTA updates at http://esp32flight.local

First boot

Tap the gear icon, scan and pick your 2.4 GHz Wi-Fi network, type the password on the on-screen keyboard and save. The device locates itself, fetches nearby flights and starts tracking. Later updates can be done over the air from the web panel.
